# Documentation

This directory contains project documentation for the assistant-api service. The structure is intentionally simple and can be expanded as the project grows.

## Sections

- **User documentation**: Installation, configuration, and operational guidance.
- **Developer documentation**: Architecture, local development, and contribution workflow.
- **AI assistant context**: Context intended for AI-based contributors, including project goals, boundaries, and terminology.

See the subdirectories for details.

## Recommended reading order

1. `docs/ai/chatgpt_context.md`: Canonical project context and constraints.
2. `docs/developer/architecture.md`: Architecture overview.
3. `docs/developer/directory_structure.md`: Directory responsibilities.
